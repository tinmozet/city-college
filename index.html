ကျွန်ုပ်တို့ ကျောင်းအကြောင်း

City College ကို ၂၀၂၀ ခုနှစ်တွင် စတင်တည်ထောင်ခဲ့ပြီး၊ ကျောင်းသား/သူများ၏ ကဏ္ဍစုံ ဖွံ့ဖြိုးတိုးတက်ရေးကို အဓိကထား သင်ကြားပေးနေပါသည်။

ကျွန်ုပ်တို့၏ သင်တန်းများ

ခေတ်မီပြီး အလုပ်အကိုင် အခွင့်အလမ်းရစေမည့် နည်းပညာသင်တန်းများ

Web Development

HTML, CSS, JavaScript တို့ဖြင့် ခေတ်မီ ဝက်ဘ်ဆိုက်များကို အခြေခံမှစ၍ စိတ်ကြိုက် ဖန်တီးနိုင်အောင် သင်ကြားပေးပါသည်။

အသေးစိတ်ကြည့်ရန်

Python & AI အခြေခံ

ယနေ့ခေတ် အသုံးအများဆုံး Python language ကို အသုံးပြုပြီး ပရိုဂရမ်ရေးဆွဲခြင်းနှင့် AI နည်းပညာ အခြေခံများကို လေ့လာရမည်။

အသေးစိတ်ကြည့်ရန်

Network Engineering

ကွန်ပျူတာ ကွန်ရက်ချိတ်ဆက်ခြင်း၊ Server ဆောက်ခြင်းနှင့် ကွန်ရက်လုံခြုံရေးဆိုင်ရာ နည်းပညာရပ်များကို လက်တွေ့သင်ကြားရမည်။

အသေးစိတ်ကြည့်ရန်

ဆက်သွယ်ရန်

သင်တန်းများနှင့် ပတ်သက်၍ အသေးစိတ် သိရှိလိုပါက အောက်ပါ လိပ်စာများမှတစ်ဆင့် စုံစမ်းမေးမြန်းနိုင်ပါသည်။
